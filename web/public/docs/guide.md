## Sharing Pastes

### Simple Share

1. Create a paste
2. Copy the URL from your browser
3. Send it to anyone

### Password Protected

1. Fill in the **Password** field
2. Create the paste
3. Others must enter the password to view

### Editable Pastes

1. Check **Allow editing**
2. Others can modify the content
3. Changes are saved immediately

## Syntax Highlighting

Binly automatically detects code syntax using **highlight.js**.

Supported languages:

- **JavaScript/TypeScript**: `javascript`, `typescript`, `jsx`, `tsx`
- **Python**: `python`
- **Go**: `go`
- **Rust**: `rust`
- **SQL**: `sql`
- **JSON**: `json`
- **Bash**: `bash`, `shell`
- **HTML/CSS**: `html`, `css`
- And 180+ more!

Manually select a syntax or let it auto-detect.

## Expiration

Set a TTL (Time to Live) for automatic deletion:

- **15 minutes**: Quick sharing
- **1 hour**: Short-term collaboration
- **1 day**: Daily notes
- **7 days**: Weekly reference
- **Never**: Permanent storage

Expired pastes are automatically deleted every 10 minutes.
